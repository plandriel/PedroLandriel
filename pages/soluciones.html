GLPI Project

GLPi es una solución libre de gestión de servicios de tecnología de la información, un sistema de seguimiento de incidencias y de solución service desk.

Este software de código abierto está editado en PHP y distribuido bajo una licencia GPL.

Zabbix

Zabbix es un Sistema de Monitorización de Redes creado por Alexei Vladishev.

Está diseñado para monitorizar y registrar el estado de varios servicios de red, Servidores, y hardware de red.

Zimbra

Zimbra es un programa informático colaborativo que está compuesto por correos electrónicos, calendarios, contactos, documentos, etc.

Es decir, se trata de un paquete de aplicaciones basado en la web y que se puede implementar como una nube privada en las instalaciones o en forma de servicio de nube pública externa.
